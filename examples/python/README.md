
#### 安裝Python
[anaconda](https://www.continuum.io/downloads)

#### 搜尋關於OpenCV的Packages
```
conda search opencv
```

#### 顯示詳細資訊
```
conda show **/**
```

#### 列出目前安裝的Package
```
conda list
```

#### 安裝OpenCV3.1
conda install -c menpo opencv3=3.1.0

#### 安裝OpenCV2.4.11
conda install -c menpo opencv=2.4.11
